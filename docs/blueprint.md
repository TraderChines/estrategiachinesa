# **App Name**: Estratégia Chinesa Pro

## Core Features:

- Headline Display: Display a prominent, responsive headline at the top of the landing page to draw attention.
- Restricted VSL Embed: Embed a streaming VSL (video sales letter) centrally on the page with player controls limited to pause only; prevent seeking.  Use the provided link.
- Timed Checkout Button: Implement a call-to-action checkout button (e.g., 'Quero Acessar o Indicador') that is initially invisible and appears automatically after 30 seconds of video playback. Links to: https://pay.kiwify.com.br/N2HRXHr
- Checkout Process: Provide a checkout process that integrates with Kiwify to handle transactions for the indicator.

## Style Guidelines:

- Background color: Dark gray (#0E0E0E) to provide a sophisticated and contrasting backdrop.
- Text color: White (#FFFFFF) for high readability against the dark background.
- Accent color: Red (#FF2A2A) for call-to-actions and highlights to draw attention.
- Font: 'Poppins', a modern geometric sans-serif, will be used for all text.
- Design a mobile-first layout optimized for vertical viewing, ensuring responsiveness across different screen sizes.
- Implement a subtle hover effect on the checkout button to indicate interactivity.
